# Personal Knowledge Vault

A landing page for a productivity app that helps people organize notes, bookmarks, and learning into structured knowledge.

> 🌱 My first project — built with pure HTML & CSS.

---

## Features

- Hero section with headline and action buttons
- Problem cards section (4 pain points)
- Solution section with feature checklist
- Early access CTA card
- Footer with social links
